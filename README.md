## 工程电磁场/电磁场

难度比较稳定，但是如果没学明白会很痛苦，可以看网课，推荐西交大马西奎的课。

南八楼下打印店似乎有这个课的题目，就叫工程电磁场，里面的题目就是仪电的，难度题型和现在考试都差不多（没记错甚至是有答案的）

## 模拟电路技术

最阴的一门课。难度忽高忽低，每年的题型难度相差都很大，而且**大部分老师几乎不捞**（所以如果有人跟你说他模电满了，很可能只是因为他那一年简单，而不是他掌握的好）

这门课很重要，也很难，是硬件的基础，短短十二周课肯定是没法掌握的，需要的是你在以后竞赛和实践中逐渐掌握三极管和一系列放大电路的用法。

如果你那年正好是模电王世隆出卷，那你就享福去吧～

## 信号与系统

这门课也很有意思，它考试和作业的题型不能说完全一样吧，只能说是毫不相关各出各的，例如最后的z变换和拉普拉斯变换，明明是期末考试的重点，作业却几乎和考试没什么关系，老师上课也不会重点讲这里。可以说这门课的难几乎是老师人为制造的壁垒，期末复习能做库里的试题就做，这一块的题目我收集的还是比较到位的。
